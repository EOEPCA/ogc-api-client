# Argo Workflows Runner APIs

Sandbox where experimenting [OGC API - Processes](https://developer.ogc.org/api/processes/index.html) OpenAPI according to [ESAEOEPCA-242](https://projects.terradue.com/browse/ESAEOEPCA-242)

## Prerequisites

In order to build the client, please make sure following tools are installed on the machine:

- [Taskfile](https://taskfile.dev/);
- [YQ](https://github.com/mikefarah/yq);
- Any [JRE](https://www.java.com/en/) >= 22.0.0.

Any other additional file will be automaticcaly downloaded during the process.

## Build

Just open the Terminal and launch

```
task build
```

the following files will be produced:

- `openapi.yaml`: the raw OpenAPI downloaded from the OGC API - Processes site;
- `ogcapi-processes.yaml`: the bundled OGC API - Processes OpenAPI (workarounds are applied);
- `ogcapi-processes-client`: the directory containing the Python client implementation.

Any other file can be ignored.
